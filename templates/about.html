ℹ️ About Project

🌍 Project Description


This project is a Machine Learning based Air Quality Prediction System developed using Python, Flask, and Multiple Classification Algorithms.

The system predicts AQI categories based on pollution parameters such as PM2.5, PM10, NO2, SO2, and CO.

🎯 Project Objectives


  • Predict AQI categories using ML
  • Compare multiple algorithms
  • Visualize pollution analytics
  • Create interactive web application
  • Improve environmental awareness

💻 Technologies Used


Technology Purpose
Python Machine Learning
Flask Backend Web Framework
HTML/CSS Frontend Design
Bootstrap Responsive UI
SQLite Database

🧠 Algorithms Used


  • Decision Tree Classifier
  • K-Nearest Neighbors (KNN)
  • Random Forest Classifier

⚙️ System Workflow


Dataset → Data Cleaning → Feature Selection → Model Training → Comparative Analysis → Best Model Selection → Flask Deployment → AQI Prediction
